🎟️ 48 Ticket Generator (Reiwa Edition)
一個結合復刻美學與現代 Web 技術的 AKB48 劇場門票產生器。支援極致自訂化、智能配色與 300 DPI 印刷品質輸出。
A premium AKB48 Ticket Generator blending retro aesthetics with modern UI/UX. Features smart contrast engine, interactive design, and high-resolution output for print.
復刻美学と現代のWeb技術を融合させた、究極のAKB48劇場チケットジェネレーター。スマート配色、インタラクティブな操作感、そして300 DPIの印刷品質に対応。
✨ 核心亮點 / Technical Highlights / 技術的な特徴
🎨 智能顯色系統 (Smart Contrast Engine)
Luminance Calculation：系統自動計算背景亮度，動態切換文字為深色或白色，並調整浮水印透明度，確保高彩度配色下的可讀性。
Dual-Color Lighting：結合成員推色 A/B，實現「雙色呼吸燈」頭像特效與主題色完美適配。
🚀 極致 UI/UX 互動 (Interactive Experience)
Glassmorphism：全站採用毛玻璃質感與動態漸變背景，營造現代沉浸感。
Expanding Nav：直覺的滑出式導航列（Expanding Pill），完美兼容電腦與手機端，提供更大的作業空間。
Contextual Editing：畫布分區 Hover 偵測，點擊門票不同位置即可精準開啟對應的抽屜式設定面板。
Global Feedback：全域點擊漣漪（Ripple Effect）與順滑的彈出式動畫。
🔗 生態圈聯動 (Ecosystem Linkage)
完美整合 AKB48 粉絲深度 MBTI 性格鑑定。
設計完門票後，系統會引導使用者前往進行與神推成員的靈魂契合度鑑定。
🌐 語言支援 / Language Support / 言語対応
自動設備語言偵測（Auto-detection for system language）
支援 7 種語言：繁體中文 / 簡體中文 / English / 日本語 / 한국어 / ไทย / Bahasa Indonesia
📝 功能說明 / Features / 機能
簡易設定 / Basic Settings / 基本設定
一鍵神推配色：內建包含 21 期研究生 在內的完整成員數據庫，選擇成員即刻完成全站配色。
即時預覽：支援 Canvas 高清渲染，縮放調整不失真，並提供互動式 Hover 反饋。
高品質匯出：支援 PNG（300DPI / 70DPI）下載，可自由開關 +3mm 出血位（Bleed）。
詳細設定 / Advanced Settings / 詳細設定
歸一化微調：精密調整文字位置（X/Y）、字間、行高、字體大小與陰影偏移。
背景深度自訂：可自由調整背景 YSS48 浮水印的偏移、間距、色彩、陰影強度與透明度。
自訂字體與圖片：支援上傳 .woff2, .ttf 等字體格式及自訂背景底圖，打造獨一無二的票券。
